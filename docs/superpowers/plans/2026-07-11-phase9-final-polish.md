# المرحلة ٩: التلميع النهائي — خطة التنفيذ

> **للمنفّذ الآلي:** SUB-SKILL: superpowers:subagent-driven-development. مربّعات `- [ ]`.

**الهدف:** إزالة آخر بقايا الأخضر البرَندي المرئي (شاشة الدخول + رؤوس أيام الجدول)، ثم فحص شامل نهائي لكل الصفحات (فاتح/داكن/جوال).

**السياق:** كل الصفحات الرئيسية والثانوية تحوّلت للنيلي (المراحل ١–٨). المتبقي المرئي: (١) شاشة الدخول (خلفية + شعار)، (٢) رؤوس أيام الجدول `.tt-day-hdr`. يوجد أخضر آخر في CSS لكنه في **كلاسات ميتة** (`.xl-*` الإكسل، `.home-stat`، `.grades-class-tab`، `.hero-card`) لم تعد تُستخدم — غير مرئية، تُترك (حذفها تنظيف اختياري لاحق).

**المعمارية:** تعديل ٣ قواعد CSS بأرقام أسطر محددة (السلاسل مكرّرة فيُستخدم sed line-targeted). لا تغيير JS.

**قرار:** توحيد ترويسة `cw-*` المكرّرة (classDetail/grades) **مؤجَّل** — إعادة هيكلة داخلية عالية المخاطر بلا فائدة بصرية؛ الصفحتان تعملان. تُترك كتحسين مستقبلي.

## Task 1: تلوين شاشة الدخول ورؤوس الجدول (CSS)

**Files:** Modify: `style.css` — ٣ قواعد (أرقام أسطر: 403 خلفية الدخول، 414 الشعار، 1151 رأس اليوم).

- [ ] **Step 1: التعديل عبر sed (استهداف أسطر محددة لتجنّب التصادم مع سلاسل مطابقة)**

نفّذ:
```bash
cd /Users/mac/Desktop/teacher
# 403: خلفية شاشة الدخول (أخضر→نيلي، مع إبقاء الكحلي النهائي)
sed -i '' '403s/#047857 0%, #065F46 60%, #1E293B 100%/#4f46e5 0%, #4338ca 55%, #1e1b4b 100%/' style.css
# 414: شعار الدخول
sed -i '' '414s/#047857, #059669/#4f46e5, #6366f1/' style.css
# 1151: رأس يوم الجدول
sed -i '' '1151s/#047857, #059669/#4f46e5, #6366f1/' style.css
```

- [ ] **Step 2: تحقّق**

```bash
sed -n '403p;414p;1151p' style.css
```
Expected: السطور الثلاثة تحتوي `#4f46e5` (لا `#047857`). و `grep -c "{" style.css` == `grep -c "}" style.css`.

- [ ] **Step 3: commit**

```bash
git add style.css && git commit -m "🎨 تلوين شاشة الدخول ورؤوس الجدول نيلي (آخر أخضر برَندي مرئي)"
```
(تجاهل تحذير git.)

## Task 2: رفع النسخة + فحص شامل نهائي

**Files:** Modify: `index.html` (النسخة).

- [ ] **Step 1:** ارفع `?v=24`→`?v=25` (style.css و app.js). commit `🔖 رفع نسخة الأصول إلى v=25`.
- [ ] **Step 2 (المنسّق):** فحص شامل بالمتصفح:
  - **شاشة الدخول** (بدون حقن): خلفية نيلية + شعار نيلي.
  - **كل الصفحات** (بحقن بيانات): الرئيسية، فصولي، مساحة الفصل (حضور/درجات/مجموعات/كتاب/طلاب)، الجدول، التحليلات، الإحالات، الإعدادات، ملف الطالب.
  - **الوضع الليلي**: تبديل وفحص عيّنة من الصفحات.
  - **الجوال**: عيّنة.
  - رصد أي أخضر برَندي مرئي متبقٍ → إن وُجد، أصلحه (سطر محدد) وأعد الفحص.
  - لا أخطاء console عبر كل الصفحات.
- [ ] **Step 3:** لقطات إثبات نهائية (الدخول + الرئيسية فاتح + عيّنة داكن + جوال).
