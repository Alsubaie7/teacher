# المرحلة ٨: الصفحات الثانوية (إحالات/إعدادات/إدارة/ملف الطالب) — خطة التنفيذ

> **للمنفّذ الآلي:** SUB-SKILL: superpowers:subagent-driven-development. مربّعات `- [ ]`.

**الهدف:** إزالة بقايا الأخضر المثبّت في الصفحات الثانوية. الفحص البصري أظهر أنها نظيفة أصلاً (نيلي) عدا ٣ مواضع: (١) بانر ملف الطالب الأخضر، (٢) رأس الجداول العامة `.data-table` (تِنت أخضر خفيف — يؤثّر على الإحالات والإدارة)، (٣) شارة الاشتراك الخضراء في الإعدادات.

**المعمارية:** تعديل ٣ قواعد CSS + سطر واحد في app.js. لا إعادة كتابة.

**التقنية:** vanilla CSS/JS. تحقّق بصري. بعد التعديل ارفع `?v=` في index.html.

## Task 1: تلوين بانر ملف الطالب ورؤوس الجداول (CSS)

**Files:** Modify: `style.css` — ٣ تعديلات.

- [ ] **Step 1: بانر ملف الطالب**

في قاعدة `.profile-banner` (حوالي السطر 1008) استبدل سطر الخلفية:
```css
  background: linear-gradient(135deg, #065f46 0%, #047857 50%, #059669 100%);
```
بـ:
```css
  background: linear-gradient(135deg, #3730a3 0%, #4338ca 50%, #4f46e5 100%);
```

- [ ] **Step 2: رأس الجدول العام**

في قاعدة `.data-table th` (حوالي السطر 642) استبدل:
```css
  background: rgba(5,150,105,.04); padding: .65rem 1rem; font-weight: 700;
```
بـ:
```css
  background: rgba(79,70,229,.04); padding: .65rem 1rem; font-weight: 700;
```

- [ ] **Step 3: تحويم صف الجدول**

استبدل قاعدة:
```css
.data-table tbody tr:hover td { background: rgba(5,150,105,.04); }
```
بـ:
```css
.data-table tbody tr:hover td { background: rgba(79,70,229,.04); }
```

- [ ] **Step 4:** `grep -c "{" style.css` == `grep -c "}" style.css`. `grep -c "rgba(5,150,105,.04)" style.css` == 0.

- [ ] **Step 5:** commit `🎨 تلوين بانر ملف الطالب ورؤوس الجداول نيلي`.
